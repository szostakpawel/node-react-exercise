### node-react-exercise

## Getting started

1. Open two tabs in IDE integrated terminal
2. In first tab move to the server directory

```
cd server
```

3. Install dependencies

```
npm install
```

4. Run the server

```
npm run dev
```

5. In second tab move to the client directory

```
cd client
```

6. Install dependencies

```
npm install
```

7. Run the app

```
npm start
```
